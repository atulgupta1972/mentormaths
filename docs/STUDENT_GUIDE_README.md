## Quick share (WhatsApp)

1. Open `docs/student-guide-whatsapp.html` in **Chrome** (double-click the file).
2. **Option A — PDF:** `Ctrl+P` → **Save as PDF** → send PDF on WhatsApp.
3. **Option B — Images:** Make browser full screen, scroll one slide at a time, screenshot each (6 slides). Send as photos in WhatsApp.

## What students learn

1. Login at **mentormaths.in**
2. **Add / edit exams** — date, chapters, optional topics
3. Find **Practice & tests · To do** on dashboard
4. **Start** → answer A/B/C/D → **Submit**
5. View score and completed sets
